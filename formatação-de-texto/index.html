As tag de Formatar textos no HTML.

Començando com negrito e italico

Essas formatações é feita com as tag Strong para o negrito (destaque) e para o italico usa a tag Em (ênfase)

Usei apenas as tag semântica da lingua

Textos Marcados

Para fazer uma marcação pique marca texto escolar usa a tag mark (envelopando)

Textos grandes e pequenos fora do css

A tag big para Textos grandes esta obsoleta e a tag small para texto pequeno pode entra em desuso logo melhor caminho para isso é usar o CSS

Texto riscado (deletados) E textos sublinhado(inserido)

Para riscar o texto com sentido de Apagado ou deletado ou algo do genero se usa a tag del, ja para sublinhar ou dar um sentido de Inserido ou espaço para complementar ou seila vamos usar a tag ins
